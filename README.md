# Employee Management System

This is a Python Flask based Employee Management System project.

## Objective
To manage employee records using a web application.

## Features
- Add employee
- View employee list
- Delete employee
- SQLite database

## Tech Stack
- Python
- Flask
- SQLite
- HTML

## How to Run
pip install flask
python app.py
Open browser: http://127.0.0.1:5000

## Conclusion
This project demonstrates CRUD operations using Python Flask.
